# Airline Services MVP
Простой MVP сервиса дополнительных услуг для авиапассажиров.  
Проект демонстрирует страницу со списком услуг, возможность добавления их в заказ и подсчет общей суммы.

## Функциональность
- Отображение списка услуг с названием и ценой.
- Кнопка «Добавить» для добавления услуги в список выбранных.
- Блок «Итого»:
  - выбранные услуги
  - общая сумма
  - кнопка «Оформить заказ» (без логики оплаты)
- Адаптивный дизайн (mobile-first).

## Технологии
- React
- TypeScript
- Функциональные компоненты
- CSS для адаптивного интерфейса

## Запуск проекта
1. Склонируйте репозиторий:

```
git clone https://github.com/ВАШ_ПОЛЬЗОВАТЕЛЬ/airline-services.git
cd airline-services
```

2. Установите зависимости:

```npm install```

3. Запустите проект:

```npm start```


Проект будет доступен по адресу http://localhost:3000.
